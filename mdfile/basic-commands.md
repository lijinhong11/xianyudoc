## /co l
> /co l 详解
有以下可用参数:
| 参数  | 说明             | 例子                |
| ------- | ------------------ | --------------------- |
| time    | 指定的时间    | time:1h               |
| radius  | 指定的查找半径 | radius:10             |
| exclude | 排除某个方块 | exclude:diamond_block |
| include | 包括某个方块 | include:iron_block    |
| action  | 执行的动作    | action:block          |
| user    | 指定玩家       | user:aaa              |
| block   | 限定作用的方块类型 | block:coal_block      |
